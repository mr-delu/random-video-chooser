# random-video-chooser
## Descrizione
Programma che, selezionati pseudocasualmente <i>n</i> elementi da una cartella contenente video e un file di soluzioni contenente le soluzioni corrispondenti ad ogni singolo video, propone i video in ordine pseudocasuale all'utente controllando per ogni riproduzione se l'input inserito è quello corretto o no.
## Funzionamento
* Eseguire il file Radom-Video-Chooser.jar
* Seguire le istruzioni a schermo
## Note aggiuntive
Nella repository sono presenti già 4 video di prova e un file con le soluzioni relative a questi. È possibile selezionare qualsiasi cartella contenente video, purchè questa non contenga altri tipi di file al suo interno. Può essere creato un nuovo file delle soluzioni o può essere modificato quello già esistente, l'importante è mantenere la stessa codifica delle righe. Le soluzioni ai video devono essere valori numerici.
